Navigation

  • handling with arrowkeys should be possible
  • footer should be configurable via options and with a full html template
  • navigation should be scrollable without scrollbar

Events

navigation.initialized

triggered when navigation is fully loaded and displayed

navigation.item.toggle[state, name, id, event]

triggered when an item with subitems is clicked

naviagtion.item.select[name, id, event, parents]

triggered when an item without subitems is clicked

Search

  • separate aura component
  • communicates with navigation via change event

Events

search.change[query] event with the query as parameter

  • triggered on key press and when query is at least three signs long
  • navigation listens for this event and filters navigation items according to query
  • filtering as full-text search

search.submit[query] event with the query as parameter

  • triggered when submit of form happend
  • relevant for searching on the right
