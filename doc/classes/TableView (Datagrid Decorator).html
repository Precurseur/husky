Parameters:

  • [viewOptions] Object optional

    Configuration object

  • [options.editable] Boolean optional

    will not set class is-selectable to prevent hover effect for complete rows

  • [options.removeRow] Boolean optional

    displays in the last column an icon to remove a row

  • [options.selectItem] Object optional

    Configuration object of select item (column)

  • [options.selectItem.inFirstCell] Boolean optional

    If true checkbox is in the first cell. If true checkbox gets its own cell

  • [options.selectItem.type] String optional

    Type of select [checkbox, radio]

  • [options.validation] Boolean optional

    enables validation for datagrid

  • [options.validationDebug] Boolean optional

    enables validation debug for datagrid

  • [options.addRowTop] Boolean optional

    adds row to the top of the table when add row is triggered

  • [options.startTabIndex] Boolean optional

    start index for tabindex

  • [options.columnMinWidth] String optional

    sets the minimal width of table columns

  • [options.fullWidth] String optional

    If true datagrid style will be full-width mode

  • [options.excludeFields=['id'] Array optional

    array of fields to exclude by the view

  • [options.showHead] Boolean optional

    if TRUE head would be showed

  • [options.icons] Array optional

    array of icons to display

  • [options.icons[] String optional

    the actual icon which sould be displayed

  • [options.icons[] String optional

    the id of the column in which the icon should be displayed

  • [options.icons[] String optional

    the align of the icon. 'left' org 'right'

  • [options.icons.callback] Function optional

    a callback to execute if the icon got clicked. Gets the id of the data-record as first argument

  • [options.hideChildrenAtBeginning] Boolean optional

    if true children get hidden, if all children are loaded at the beginning

  • [options.openChildId] String | Number | Null optional

    the id of the children to open all parents for. (only relevant in a child-list)

  • [options.cssClass] String | Number optional

    css-class to give the the components element. (e.g. "white-box")

  • [options.highlightSelected] Boolean optional

    highlights the clicked row when selected

  • [rendered] Boolean optional

    property used by the datagrid-main class

  • [initialize] Function optional

    function which gets called once at the start of the view

  • [render] Function optional

    function to render data

  • [destroy] Function optional

    function to destroy the view and unbind events

  • [onResize] Function optional

    function which gets automatically executed on window resize

  • [unbindCustomEvents] Function optional

    function to unbind the custom events of this object
