Constructor

Dropzone

(
  • [options]
)

Defined in husky_components/dropzone/main.js:12

Parameters:

  • [options] Object optional

    Configuration object

    • [titleKey] String optional

      The title to display if no items are dropped (can be a translation key)

    • [descriptionKey] String optional

      The description to display if no items are dropped (can be a translation key)

    • [descriptionIcon] String optional

      The icon to display if no items are dropped

    • [cancelLoadingIcon] String optional

      the icon which gets displayed while a file item gets uploaded

    • [method] String optional

      method to use for uploading 'PUT' or 'POST'

    • [url] String optional

      url to upload the files to

    • [paramName] String optional

      the name of the parameter which will contain the file(s). Note that if uploadMultiple is set to true the parameter Name gets extended with '[]'

    • [headers] Object optional

      additional headers to pass with each request

    • [uploadMultiple] Boolean optional

      if true a request can upload multiple files

    • [maxFiles] Number optional

      defines the maximum of files in the dropzone

    • [successCallback] Function optional

      callback which gets called if a file got successfully uploaded. First parameter is the file, the second the response

    • [beforeSendingCallback] Function optional

      callback which gets called before a file gets uploaded. First parameter is the file.

    • [removeFileCallback] Function optional

      callback which gets called after a file got removed. First parameter is the file.

    • [pluginOptions] Object optional

      Options to pass to the dropzone-plugin to completely override all options set by husky. Use with care.

    • [showOverlay] Boolean optional

      if true the dropzone will be displayed in an overlay if its not visible any more or the passed scroll-top is reached

    • [skin] String optional

      skin class for the dropzone. currently available: 'small' or '' (default)
